# [23. Merge k Sorted Lists](https://leetcode.com/problems/merge-k-sorted-lists/)

给你一个链表数组，每个链表都已经按升序排列。

请你将所有链表合并到一个升序链表中，返回合并后的链表。

<b>示例1</b>
```
输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：[1,1,2,3,4,4,5,6]
解释：链表数组如下：
[
  1->4->5,
  1->3->4,
  2->6
]
将它们合并到一个有序链表中得到。
1->1->2->3->4->4->5->6
```
<b>示例2</b>
```
输入：lists = []
输出：[]
```
<b>示例3</b>
```
输入：lists = [[]]
输出：[]
```

<b>解题方法</b>
* 方法一 
> 合并多个链表 &rarr; 反复合并两个链表。所以该题只需要实现如何合并两个有序链表+如何反复该过程即可。

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeKLists(self, lists: List[ListNode]) -> ListNode:
        n = len(lists)
        
        if n==0:
            return None
        if n==1:
            return lists[0]
            
        while n>1:
            k = (n+1)//2
            for i in range(n//2):
                lists[i] = self.mergeTwoLists(lists[i], lists[i+k])
            n = k
        return lists[0]
    
    # merge two linked lists
    def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
        dummy = ListNode()
        p = dummy
        
        p1, p2 = l1, l2
        
        while p1 and p2:
            if p1.val<p2.val:
                p.next = p1
                p1 = p1.next
            else:
                p.next = p2
                p2 = p2.next
            p = p.next
            
        if p1:
            p.next = p1
        
        if p2:
            p.next = p2
        return dummy.next
```

* 方法二

> 由于每次要从多个list中挑选最小的那个，可以使用<u>堆heap</u>来实现。每次将堆顶弹出作为新节点，再将该堆顶元素所属list的下一个值放入堆中，并不断重复该过程。

```py
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeKLists(self, lists: List[ListNode]) -> ListNode:
        n = len(lists)
        if n==0:
            return None
        if n==1:
            return lists[0]
        
        import heapq
        dummy = ListNode(0) # dummy node
        p = dummy           # 逐步生成链表的指针
        heap = []           # 存放比较元素的堆
        heapq.heapify(heap) # 堆化
        
        # 初始化堆
        for i in range(n):
            if lists[i]:
                heapq.heappush(heap, (lists[i].val, i))
                lists[i] = lists[i].next
        
        # 处理后续元素，只到堆里没有元素可以比较
        while heap:
            val, idx = heapq.heappop(heap)  # 获取最小的值对应的值和所属的list
            p.next = ListNode(val)      # 创建新的node
            p = p.next                  # 等待获取下一个node
            if lists[idx]:              # 如果该list还有元素，则放入
                heapq.heappush(heap, (lists[idx].val, idx))
                lists[idx] = lists[idx].next
        return dummy.next
```
